# Aero Weather Dashboard

A minimalist weather application designed to give you exactly what you need to know at a glance, without the clutter of traditional weather sites.

## Features
* **Geolocation:** Automatically detects your current city.
* **Interactive Forecast:** A 7-day sliding forecast window.
* **Dynamic Backgrounds:** The UI theme shifts based on the current weather conditions (e.g., dark grays for rain, bright gradients for clear skies).

Data is fetched using the OpenWeather API.
