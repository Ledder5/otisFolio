# ZenTask

A lightweight alternative to Jira and Trello, built specifically for solo developers managing side projects.

## Tech Stack
* **Frontend:** Vue 3 with the Composition API for reactive state management.
* **Styling:** Tailwind CSS for a clean, customized UI.
* **Backend/Auth:** Firebase Authentication and Firestore for real-time database syncing across devices.

The drag-and-drop functionality was custom-built using standard HTML5 Drag and Drop APIs to keep the bundle size small.
